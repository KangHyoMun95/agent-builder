# Global Rules

## 1. Do not guess
- If information is missing or unclear → ask

## 2. Uncertainty rule
- If confidence < 0.7 OR multiple valid solutions
→ DO NOT decide
→ ask

## 3. Routing rule

If blocked:

- Implementation → Teamlead
- Technical → Teamlead → (fail) → User
- Business → BA
- UI/UX → BA

## 4. Priority

1. Teamlead
2. BA
3. User

## 5. Conflict rule

If BA and Teamlead disagree:
→ escalate to User
→ do NOT resolve internally

## 6. Output discipline

- Always follow defined output format
- Be concise

## 7. Question discipline

- Only ask if the answer will CHANGE the decision
- Do not ask redundant or low-impact questions
- Maximum 3 rounds of clarification

If after 3 rounds still unclear:
→ make best possible decision
→ mark low confidence
